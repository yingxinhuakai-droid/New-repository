教研团队 AI 助手使用培训手册

从零开始，15 分钟学会用标准化提问让 AI 替你产出讲义、题目、课件、解析和报告。

📅 2026 年 6 月 👥 面向全体教研老师 ⏱ 阅读约 12 分钟
Part 1

快速入门：AI 能帮教研做什么？

先搞清楚 AI 的能力边界，才知道什么时候该用它、怎么用最值。

教研场景AI 能做什么你不能期待 AI 做什么
讲义编写 按考纲章节生成讲义初稿、考点清单、背诵手册，教研审核后定稿 100% 无误的终稿（必须人工核对法条编号和计算公式）
题目命制 根据考点生成单选题、多选题、案例分析题 + 完整解析 + 法条依据 直接上架的成品题（需要教研逐题审核难度和合规性）
课件制作 输出 PPT 逐页内容框架、讲解逻辑、随堂提问设计 直接可用的 .pptx 文件（需要手动或通过腾讯文档套入模板）
答疑解析 生成标准答案 + 法条依据 + 计算步骤 + 得分框架 代替教研老师的专业判断（需教研确认答案口径）
数据分析 根据你给的错题数据表格，产出错误类型分类、薄弱考点追溯、TOP 榜单 无数据凭空分析（必须提供原始数据）
考纲研读 新旧考纲逐条对比、标注增减和分值变化，输出对比报告 预测出题方向（只能基于考纲文本分析，无法替代教研的经验判断）

核心原则：AI 是"初稿机器"，教研是"质量守门人"

你的价值在于审核、校准、拍板。AI 帮你省掉 80% 的重复性初稿劳动，剩下 20% 的关键决策必须你来做。

Part 2 · 核心

万能提问模板：五要素法

这是本手册最重要的内容。无论什么教研任务，套这个模板，输出质量立刻提升一个档次。

⭐ 收藏这个模板
标准化提问五要素
缺一个要素，AI 就可能自由发挥，结果不可控
1
角色设定 —— 告诉 AI 它现在是谁
示例：「你是一位有 10 年教龄的中级注册安全工程师培训讲师」
2
任务描述 —— 具体要做什么事
示例：「请编写一份《安全生产法律法规》第三章的考点清单」
3
输入材料 —— 给 AI 提供原始素材
示例：「以下是本章官方考纲原文：[粘贴内容]」或上传考纲文件
4
输出要求 —— 格式、层级、长度、风格
示例：「按掌握/熟悉/了解三级分类，每个考点标注法条编号和近 3 年考频，输出为表格格式」
5
约束条件 —— 明确不要什么
示例：「不要展开超纲内容，不要使用非官方术语，法条必须精确到条款号」

把五要素浓缩成一句话模板

【复制下面这段话，填空即可】 你是一位 [角色，如：注安法规科目资深讲师]。 请帮我 [任务，如：编写技术科目第三章讲义初稿]。 以下是参考材料：[粘贴考纲/教材原文/已有资料]。 请按以下要求输出： 1. [格式要求：如表格/分点/Markdown] 2. [内容要求：如分层级/标注考频/含法条依据] 3. [长度要求：如每章不超过 2000 字] 注意：[约束条件：如不要超纲/不要口语化/法条必须精确]
Part 3 · 实战

六大场景 · 完整案例演示

每个案例都展示"小白常犯的模糊提问"和"标准化提问"，对照学习效果最好。

1

讲义编写：安全生产法律法规考点清单

教材研发

帮我写一份安全生产法的考点清单。

👆 这种写法AI不知道你要哪个章节、什么层级、什么格式，出来的东西大概率不能用。

你是一位有 8 年教龄的中级注册安全工程师《法规》科目培训讲师。

请帮我编写《安全生产法律法规》科目中"第三章 安全生产法"的考点清单，面向基础阶段学员。

以下是本章考纲原文：
[此处粘贴 2025 版中级注安法规科目第三章考纲内容]

请按以下要求输出：
1. 按"掌握 / 熟悉 / 了解"三级分类
2. 每个考点标注对应法条编号（如《安全生产法》第 XX 条）
3. 标注近 3 年（2023-2025）该考点在真题中的出现频次
4. 每个考点附加一句"易错提醒"
5. 输出为 Markdown 表格格式

注意：
- 以 2021 年修订版《安全生产法》为法条依据
- 不要引入其他单行法规的内容
- 考点描述控制在 30 字以内

这个提问为什么好？

  • ✓ 角色明确：法规定向讲师，AI 会自动调用法规知识
  • ✓ 层级清晰：基础阶段，AI 知道不写拔高内容
  • ✓ 输入具体：粘贴考纲，AI 有据可依
  • ✓ 格式固定：表格 + 三级分类 + 考频 + 易错提醒，输出结构完全可控
  • ✓ 有约束：限定了法条版本、不超纲、字数控制
2

题目命制：原创单选 + 完整解析

题库建设

帮我出几道关于电气安全的单选题。

你是一位中级注册安全工程师《安全生产技术基础》科目的命题专家。

请根据以下考点出 3 道单选题：
考点：电气安全技术 — 触电防护技术 — 保护接地与保护接零
难度：中等（适合已完成基础阶段学习的学员）

参考考纲要求：
[此处粘贴电气安全章节考纲]

请按以下要求输出每道题：
1. 题干：场景化描述（贴近实际工作场景），50-80 字
2. 四个选项（A/B/C/D），干扰项要有迷惑性（常见概念混淆点）
3. 正确答案 + 完整解析（含：法条/标准依据、判断逻辑、每个错误选项的排除理由）
4. 标注考点对应的考纲编号
5. 附加：该题的"学员常见错误思路"分析

注意：
- 依据 GB/T 13869《用电安全导则》和《安全生产技术基础》官方教材
- 不要出超纲或已废止标准的题目
- 选项不要出现"以上都对/以上都不对"

关键技巧

  • ✓ 要求场景化题干 + 有迷惑性的干扰项 → 出题质量接近人工
  • ✓ 要求标注考纲编号 → 方便后续题库标签管理
  • ✓ 要求"学员常见错误思路" → 直接可用于答疑和磨课
  • ✓ 约束"不出超纲题、不出以上都对" → 避免 AI 偷懒
3

课件制作：一节课的完整教案 + 逐页 PPT 内容

课程设计

帮我做一个安全管理课程的 PPT。

你是一位中级注安《安全生产管理》科目的资深讲师，擅长将复杂知识点拆解为易于讲解的课件结构。

请为以下知识点制作一节课（45 分钟）的完整教案和 PPT 内容框架：
知识点：安全生产管理 — 安全评价方法 — 安全检查表法（SCL）
授课对象：零基础学员（第一次接触安全评价方法）

参考材料：
[粘贴教材对应章节内容]

请输出：

第一部分：教案
1. 教学目标（知识目标 + 能力目标）
2. 重难点分析（标注"重点""难点"）
3. 教学时间分配（导入 5min / 讲解 25min / 案例 10min / 小结 5min）
4. 与本课程其他知识点的衔接说明

第二部分：PPT 逐页内容（共约 15-18 页）
每页包含：
- 页面标题
- 核心内容（3-5 个要点）
- 讲师备注（这一页该怎么说、什么语气、要不要互动）
- 如有案例，完整写出案例描述和分析

第三部分：随堂互动设计
- 3 个课堂提问（含参考答案）
- 1 个小组讨论题目

注意：
- 用生活化比喻辅助讲解专业概念（如把 SCL 比作"出门前的物品清单"）
- 避免大段文字堆砌，PPT 每页要点不超过 5 条
- 语言口语化但不失专业感

这个案例的亮点

  • ✓ 一次性产出教案 + PPT 框架 + 互动设计，三合一
  • ✓ "生活化比喻"的要求，让零基础学员听得懂
  • ✓ 明确了"讲师备注"和"互动设计"，不只是给内容还给教法
4

答疑解析：一道案例题的标准答案 + 得分框架

答疑标准化

这道案例分析题答案是什么？[粘贴题目]

你是一位中级注册安全工程师《安全生产专业实务》科目的阅卷专家，熟悉主观题的评分标准和踩分规则。

请为以下实务案例分析题生成标准答案和评分细则：
[粘贴完整的案例题目，包含背景材料和小问]

请按以下结构输出：
1. 考点识别：列出本题涉及的考点及考纲编号
2. 解题思路：分步骤说明这道题的思考路径
3. 标准答案：逐问给出完整参考答案
4. 评分细则：每小问的踩分点、分值分配、得分关键词
5. 常见失分点：学员答这道题最容易在哪里丢分，为什么
6. 答疑引导话术：如果学员答错了，老师应该怎么引导他找到正确答案（不要直接说答案）

注意：
- 法条引用必须精确到条款号
- 评分细则要细化到"写出XX给 2 分，只写YY 给 1 分"的粒度
- 答疑话术要"启发式"，不是"告知式"

这解决了什么痛点？

  • ✓ 四位老师讲同一道题，答案和评分标准完全一致
  • ✓ 答疑不再靠老师个人发挥，有标准引导话术
  • ✓ 评价细则可直接用于模考阅卷

考纲分析：新旧考纲对比报告

内容生产

帮我对比一下新旧考纲有什么变化。

你是一位中级注册安全工程师考试政策研究专家，每年第一时间研读官方考纲变化。

请对比以下两份考纲，输出差异分析报告：
旧版：2024 版中级注安《安全生产技术基础》考试大纲
[粘贴 2024 版考纲全文或上传文件]
新版：2025 版中级注安《安全生产技术基础》考试大纲
[粘贴 2025 版考纲全文或上传文件]

请按以下结构输出报告：
1. 总体变化概述（100 字以内总结）
2. 新增考点清单（逐条列出，标注所在章节）
3. 删除考点清单（逐条列出，标注原所在章节）
4. 调整考点清单（原考点内容有变化的，标注变化细节）
5. 分值权重变化分析（如考纲有标注分值变化的）
6. 对教研资料的影响评估：哪些讲义/题目/课件需要更新
7. 建议更新优先级（紧急 / 重要 / 可延后）

注意：
- 逐条对比，不要遗漏任何细节
- 对"措辞微调但实质未变"的考点特别标注
- 输出为正式报告格式，语体书面化

数据分析：学员错题诊断报告

质量管控

帮我分析一下学生的错题数据。

你是一位教育数据分析师，擅长从学员答题数据中诊断教学薄弱环节。

以下是某期注安《管理》科目模考错题数据（共 200 名学员参加）：
[粘贴错题数据表格，含：题号、考点名称、错误人数、错误率、常见错误选项]

请按以下要求产出分析报告：
1. 错题 TOP 10 榜单（按错误率降序）
2. 错误类型分类统计：
- 法条混淆类（选了相似法条的干扰项）占比 ___%
- 概念不清类（选了概念相近的错误选项）占比 ___%
- 计算失误类（计算题选错）占比 ___%
- 审题偏差类（题目理解错误）占比 ___%
3. 薄弱考点集群：按章节汇总失分率，识别最薄弱的 3 个章节
4. 教学改进建议：针对每个薄弱考点，给出具体的补漏方案（重讲知识点 / 出变式题 / 录制微课）
5. 学员分层建议：按分数段划分零基础/中等/拔高，给出针对性学习建议

注意：
- 所有百分比保留一位小数
- 改进建议要具体到"XX 章节 XX 知识点需要补充 XX 类型练习"
- 报告语言要教研向（可被教学主管直接用于教研会讨论）

Part 4

进阶技巧：让 AI 输出更精准

掌握了五要素模板后，这些技巧能让你的效率再翻倍。

📎

上传附件代替粘贴

考纲 PDF、教材 Word、错题 Excel 都可以直接上传文件，比手动粘贴省时间，而且 AI 能读取表格和 PDF 中的结构化数据。

🔄

不满意就追问

AI 一次输出不够好？直接说"第三题难度太低，重出""考点清单太简略，每个考点加 50 字解释"。追问不花钱。

📋

批量处理

别一次只让 AI 做一道题。说"请按上述模板出 10 道题"，一次产出，逐个审核，效率高得多。

🔗

上下文接力

上一轮生成了讲义，下一轮可以说"基于刚才的讲义，帮我出配套练习题"。AI 记得上下文，不用重复粘贴。

📊

直连腾讯文档

生成的表格、讲义、报告可以直接写入腾讯文档在线协同编辑，不需要复制粘贴切换工具。

🤖

设置定时任务

每周一自动抓取应急管理部最新通知生成简报、每月 1 号自动产出上月错题报告。设置一次，自动执行。

Part 5

新手最容易犯的 6 个错误

避开这些坑，输出质量立刻提升。

  • ✕
    只有任务，没有角色。
    错误：「帮我写一份讲义。」
    正确：「你是一位注安技术科目讲师，请帮我写电气安全章节讲义。」
    原因：没有角色，AI 不知道用什么知识体系和语言风格。
  • ✕
    没有给参考材料。
    错误：「帮我出一道关于应急预案的题。」
    正确：粘贴考纲原文或教材内容后再请 AI 出题。
    原因：不给材料，AI 可能引用过时或错误的标准，你得花更多时间核对。
  • ✕
    没有指定格式。
    错误：「写个考点总结。」
    正确：「用 Markdown 表格输出，三列：考点名称 / 法条依据 / 考频。」
    原因：AI 默认输出可能是段落文字，不是你想要的表格或清单。
  • ✕
    没有设置约束条件。
    错误：「帮我写一份安全管理的讲义。」
    正确：「…注意：以 2021 版《安全生产法》为准，不要引入已废止的 2014 版内容，不要超纲。」
    原因：不设边界，AI 可能混入旧法条或拓展到不相关领域。
  • ✕
    一次塞太多任务。
    错误：「帮我写安工四科的讲义、题库、课件、还有学习计划。」
    正确：一次聚焦一个任务，完成后用上下文接力做下一个。
    原因：任务太多 AI 容易混淆重点，每个任务的输出质量都会下降。
  • ✕
    拿到输出不审核直接用。
    错误：AI 出的题直接录入题库。
    正确：逐题核对法条编号、确认考纲匹配、检查选项逻辑，确认无误后入库。
    原因：AI 不能替代专业判断，法条编号和标准引用是教研的生命线。

快速参考卡：每个场景的提问关键词

不记得完整模板时，打开这张卡，找到对应场景，用关键词拼出你的提问。

教研场景 × 提问关键词速查

讲义/考点清单
角色：XX科目讲师 + 章节 + 基础/提升/冲刺 + 考频标注 + 法条依据 + 表格格式
出题
角色：命题专家 + 考点 + 难度 + 题型 + 场景化题干 + 干扰项迷惑性 + 完整解析 + 考纲编号
课件/教案
角色：XX科目讲师 + 知识点 + 学员水平 + 教案(目标/重难点/时间) + PPT逐页内容 + 讲师备注 + 互动设计
答疑解析
角色：阅卷专家 + 考点识别 + 解题思路 + 标准答案 + 评分细则 + 常见失分点 + 引导话术
考纲对比
角色：政策研究专家 + 新旧考纲全文 + 新增/删除/调整清单 + 分值变化 + 影响评估 + 更新优先级
错题分析
角色：数据分析师 + 原始数据表格 + TOP榜单 + 错误分类统计 + 薄弱章节 + 补漏建议 + 学员分层
学习计划
角色：备考规划师 + 学员基础类型 + 备考周期 + 四科搭配 + 每周任务 + 阶段目标 + 时间表格
速记/口诀
角色：记忆法专家 + 知识点列表 + 口诀要求(押韵/谐音/联想) + 附带解释说明
Bonus

发送前自查清单

每次提问前，花 10 秒钟过一遍，确保不踩坑。

  • ✓ 我设定了角色吗？（AI 知道它是谁）
  • ✓ 任务描述具体吗？（不是"帮我写个东西"）
  • ✓ 给了参考材料吗？（考纲/教材/数据表格）
  • ✓ 指定了输出格式吗？（表格/分点/报告/PPT框架）
  • ✓ 设置了约束条件吗？（法条版本/不超纲/字数限制）
  • ✓ 一次只做一个任务吗？（不贪多）
