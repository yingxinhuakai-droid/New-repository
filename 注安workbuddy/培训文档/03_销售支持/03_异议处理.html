❓ 异议处理手册（10 大抗拒点）

客户说的每句"不"背后都有原因——Q&A + 心理学 + 应对话术

一、万能公式 LAMP（背熟就能应对 80% 异议）

🧠 LAMP 四步法

L - Listen 倾听：让客户说完，不打断，不反驳
A - Acknowledge 共情："我理解您的顾虑..." / "确实很多人一开始也这么想..."
M - Make it right 解决：给具体方案（FAB 话术 + 证据）
P - Probe 追问："这样您看可以吗？" / "您还有什么顾虑吗？"

二、10 大抗拒点速查（点击跳转）

1. 太贵了
2. 没时间学
3. 考了也没用
4. 我考虑下
5. 别的机构更便宜
6. 通过率真有那么高？
7. 怕被坑
8. 退费麻烦
9. 家人不支持
10. 我二战了，怀疑自己
抗拒 1：太贵了，能便宜点吗？
🧠 客户心理：不是真嫌贵，是没看到"非你不报"的理由 + 想试探议价空间
应对话术（价值塑造 + 拆解单价）

👉 先共情："理解您觉得一次性投入不小，我帮您算笔账..."

👉 再拆解："云私塾 ¥2,999，平均每天 8 块钱，比一杯奶茶便宜。算下来 1 科不到 ¥750，比单科报班划算 30%。"

👉 给方案："3 人成团 5 折，人均 ¥1,500。协议班不过退 70%，等于不亏。"

👉 最后追问："这样看，您还觉得贵吗？"

❌ 禁忌

"一分价钱一分货" / "我们就是贵" / 直接降价（失去信任）

抗拒 2：没时间学 / 我工作太忙
🧠 客户心理：真实痛点 + 需要被理解 + 想要"时间少也能过"的方案
应对话术（场景化 + 工具替代）

👉 先共情："您这岁数正是事业爬坡期，时间确实金贵。"

👉 再方案："我们云私塾 AI 智能题库，错题自动归类，每天 30 分钟 就能完成 1 章节。私教 max 1v1，午休 10 分钟答疑 就行。"

👉 给数据："去年 35% 学员是上班族，6 科平均 4 个月拿证。"

👉 反问："您每天通勤 + 睡前 1 小时，加起来 90 分钟，够吗？"

❌ 禁忌

"您再忙也要学啊" / "别人都能学" / 否定客户时间

抗拒 3：考了也没用 / 不想考
🧠 客户心理：往往是被动咨询（公司要求 / 家人劝），内心抗拒
应对话术（价值重塑 + 政策利好）

👉 先共情："您这想法很真实，很多人一开始也这么想。"

👉 算账："中级注安证对应中级职称，公司一般给 ¥800-2000/月补贴，一年回本。"

👉 政策："26 年应急管理部新规，高危行业必须配齐注安师，以后会越来越严。"

👉 反问："您身边同事有在考吗？大家都在为这事布局，您看呢？"

抗拒 4：我考虑下 / 再看看
🧠 客户心理：真考虑（30%）+ 找借口推脱（70%）。要识别真假。
应对话术（探真伪 + 紧迫感）

👉 先探："完全理解，考证是大事。您主要是哪方面还在考虑？价格 / 时间 / 效果？"（让他说出具体顾虑）

👉 看反应：能说出具体顾虑 → 真考虑，给方案；含糊其辞 → 假推脱

👉 真考虑：给限时优惠："3 人成团 5 折 6.18 截止，要不我帮您留个位？"

👉 假推脱：给台阶："您方便留个微信，我把资料发您，1 周后我联系您？"（不丢失）

❌ 禁忌

强逼："今天必须报" / 客户已说"考虑"还继续推

抗拒 5：别的机构更便宜 / 233 才 2200
🧠 客户心理：在比价，需要"贵得有理由"
应对话术（差异化 + 真实对比）

👉 承认差异："是的，233 基础班 ¥2,200，我们贵 35%。但您看这 35% 差在哪——"

👉 对比表（直接发销售工作台链接）：233 是录播 + 大班，我们是 AI 智能 + 1v1 私教 + 不过退 70%。

👉 3 人团价格："我们云私塾 3 人成团 5 折，人均 ¥1,500，比 233 还便宜 30%。"

👉 反问："您是单纯比价，还是希望真正一次过？"

抗拒 6：通过率真有那么高吗？
应对话术（数据 + 案例 + 兜底）

👉 给数据："环球网校 25 年通过率 78%，协议班通过率 92%。"

👉 给案例："您看这位 XX（保定的李姐），38 岁宝妈，去年报的云私塾，6 科一次过。"

👉 给兜底："而且我们协议班不过退 70%，考不过也不亏。"

抗拒 7：怕被坑 / 你们会不会跑路？
应对话术（品牌 + 资质 + 退费承诺）

👉 品牌背书："环球网校 2003 年成立，新东方旗下，上市公司，跑不了。"

👉 资质："我们协议班白纸黑字，签合同，不过退 70%。"

👉 反问："您之前有被坑过吗？哪些点比较担心？我重点说。"

抗拒 8：退费麻烦 / 万一不退呢？
应对话术（流程化 + 兜底）

👉 流程："我们退费是 T+15 工作日，合同写清楚，原路退回。"

👉 兜底："25 年协议班 0 投诉，您可以查。"

👉 反问："您最担心退费哪个环节？"

抗拒 9：家人不支持 / 老婆不同意
应对话术（拉家人 + 算账）

👉 算账："您算一下，拿证后补贴 1500/月，一年 1.8 万，云私塾 2999，不到 2 个月回本。"

👉 拉家人："要不嫂子也一起听下？我们 3 人成团 5 折，两个人一起考更划算。"

抗拒 10：我二战了，怀疑自己...
🧠 客户心理：心理创伤 + 自我怀疑 + 急需被鼓励
应对话术（共情 + 找原因 + 方案）

👉 先共情："二战不容易，您能坚持本身就了不起。"

👉 找原因："您去年哪科没过？是复习没到位，还是时间不够？"

👉 给方案："二战最关键的是找对方法 + 跟对人。我们私教 max 1v1，针对您弱项专项突破，不过退 70%。"

👉 给数据："去年二战学员通过率 89%，三战才掉到 60%。"
