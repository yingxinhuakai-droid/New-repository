🎯 直播定位

  • 主推产品：云私塾正价课 / 云私塾定金 9 元

🧠 四、5 条可复用的直播方法论

方法论 1·低定金钩子：9 元定金 = 筛选真正有意向的潜在客户 + 沉淀私域。9 元定价是经过测试的"心理门槛"。

方法论 2·产品 + 嘉宾双 IP：直播不只是卖产品，是卖老师 + 卖课程 + 卖方法论的组合。嘉宾 IP 决定信任度，产品 IP 决定转化率。

方法论 3·直播 + 二次转化：直播中没买的人，不是"无意向"，而是"还没准备好"。所以必须有 直播后 D+1 / D+3 / D+7 三波推送，才能把流量榨干。

方法论 4·物料清单化：物料清单要前置 7 天准备好，包括 PPT / 海报 / 视频 / 抽奖道具 / 客服话术 / 推送短信 6 大类。

方法论 5·标签驱动复购：直播打的所有用户标签 = 第二波销售的弹药库。标签越精细，复购率越高。
