模块五·考前冲刺服务

45天冲刺方案定制→集中答疑预判→考前心理建设。让学员带着信心走进考场。含3个场景+1个动手任务。

工作场景

距离注安考试还有45天。学员开始两极分化：学得好的更努力了，学得不好的开始摆烂。你需要在有限的时间内帮每个人制定冲刺方案——但每个人的基础不同、薄弱科不同、可用时间不同，你不可能给170人每人定制一份。同时学员的焦虑感在上升，每天都有人问"老师我还能过吗"。

角色：班主任 · 冲刺阶段的总调度

核心痛点

  • 01冲刺方案没法批量定制——每个学员四科水平不同，A学员法规强实务弱、B学员反过来，不能用同一套方案
  • 02不知道帮学员"抢分"抢哪里——时间有限，胡子眉毛一把抓一定不行，需要精准判断"学哪里投入产出比最高"
  • 03考前学员情绪脆弱——一句话说不好可能让学员彻底放弃
  • 04答疑预判不足——考前学员会集中问很多问题，来不及准备答案

提示词模板

模板 5.1：45天冲刺方案
学员[姓名]，[科目组合]。
各科水平：法规[分/水平]，管理[分/水平]，技术[分/水平]，实务（[方向]）[分/水平]。
每天可用：[小时]（[白天/晚上/周末]），距考试45天。
帮我做冲刺方案：
①四科时间分配（按提分空间排序，不要平均）
②前15天主攻什么→中间15天→最后15天各阶段策略
③每个阶段结束后该达到什么水平
④今天就能执行的第一个动作
输出成表格+说明。
角色 班主任的AI助手 任务 制定个性化冲刺方案 格式 表格+说明 约束 时间分配不平均，按提分空间排序 输出 三阶段策略+今日可执行动作
模板 5.2：考前答疑预判
我带的是中级注安班，学员主要薄弱点在[列举]。
距考试还有[天数]，帮我预判考前最可能被问到的20个问题：
- 备考方法类（5个）
- 法规难点（5个）
- 实务答题技巧（5个）
- 考试当天注意事项（5个）
每个问题给简洁版答案（50字以内，能直接回微信的）。
角色 班主任的答疑预判助手 任务 预判高频问题+备好答案 格式 四类各5个，共20题 约束 答案50字以内，可直接复制 输出 问答清单，微信可直接发
模板 5.3：考前心理建设
帮我设计考前心理建设话术，分三个节点：
①考前7天：稳定心态 + 提醒不要学新东西
②考试当天早上7点：简短有力，提振信心
③考后（不管考得怎样）：三种情况各一条（考得好/一般/感觉不好）
每条50字以内，有温度但不说空话。
角色 班主任的心理支持顾问 任务 设计三节点心理建设话术 格式 三个节点×不同话术 约束 每条50字内，有温度不说空话 输出 可直接发送的微信话术

案例演示

案例 A：偏科学员冲刺方案
Input
学员程甫, 建筑全科, 法规78/管理65/技术42/实务建筑38, 每天2小时晚上
Output

时间分配：技术40% + 实务25% + 管理20% + 法规15%（法规已稳，维持即可）

前15天：主攻技术科目（提分空间最大），技术和实务合计占60%时间。每天1.2h技术+0.8h实务案例。

中间15天：技术继续巩固+加入法规管理真题巩固。法规每天只做10道题保持手感。

最后15天：四科真题轮测+模拟考试，查漏补缺。

今日第一动作：打开技术科目第三章，做15道课后练习题，记录错题。

案例 B：考前答疑预判清单
Input
中级注安班，薄弱点：实务案例、技术计算、法规细节
Output

20个问题覆盖四科+技巧+心态，每个50字内简洁版答案。班主任可以直接复制粘贴用。

示例——备考方法类Q1："最后45天只刷题行不行？" → 不行，刷题不回顾=白刷。建议：每天做题后花20分钟回顾错题知识点，比多刷50道更有效。

示例——法规难点Q1："安全生产法和管理条例怎么区分？" → 看发布机关：人大是法，国务院是条例。罚款金额记法里的，管理条例考程序。

模块六·续报与转介绍

判断续报时机→不反感的续报话术→三版转介绍邀请。让续报成为服务闭环的自然结果而非强行推销。含2个场景+1个动手任务。

工作场景

学员今年报了2科（法规+管理），付费3980元。法规学得很好（正确率72%），管理也还行（58%）。距离考试60天。作为班主任你知道——如果他今年只过2科，明年还得报另外2科，但你现在提续报，会不会让他觉得"你只关心我交钱不关心我学"？什么时候提、怎么提，这是艺术。

角色：班主任 · 续报转化操盘手

核心痛点

  • 01把握不好提续报的时机——提早了显得功利，提晚了学员自己找别家了
  • 02不会自然引入——"你再报两科吧"太生硬，但不知道怎么说才不反感
  • 03转介绍不好意思开口——明明服务做得不错，但让学员"帮忙介绍朋友"说不出口
  • 04不同满意度学员该说不同话——铁粉可以直说，一般满意的不敢提，但又想提

提示词模板

模板 6.1：续报时机判断
学员[姓名]，报了[科目]，付费[金额]。
各科进度和正确率：[描述]
距考试[天数]。他的态度：[描述]
帮我分析：
①现在该不该提续报？为什么？
②如果现在不提，最佳时机是什么？
③如果现在可以侧面提，怎么说（不要直接说"再报两科"）
④输出一条今天可以发的、植入续报暗示但不引起反感的消息
角色 续报策略顾问 任务 判断时机+设计植入话术 格式 四步分析+一条可直接发的消息 约束 不能直接说"再报两科" 输出 时机判断+植入消息
模板 6.2：转介绍邀请
学员[姓名]，[科目]，各科成绩[描述]，对服务满意度[描述]。
帮我生成转介绍话术，分三个版本：
①铁粉版（学员特别满意，可以直说）
②满意版（学员觉得还行，但不太主动）
③一般版（学员没特别满意但也没不满，试探性提）
每条控制语气，不要像群发广告。
角色 转介绍话术设计师 任务 生成三档转介绍话术 格式 铁粉/满意/一般三个版本 约束 不要像群发广告，语气自然 输出 三条可直接发送的微信话术

案例演示

案例 A：今年只报2科的学员，现在提续报吗？
Input
法规72%管理58%, 距考试60天, 态度"先考过这两科明年再说"
Output

判断：现在不适合直接提续报。学员态度明确"先考过再说"，硬推会触发反感。

最佳时机：考完后根据成绩提——如果他法规考得好、管理擦线过，那时提"要不要趁热打铁把剩下两科也拿下"成功率最高。

今天可发的植入消息："你这法规正确率稳在72%，管理再冲一冲也很有希望。如果你今年两科都过了，剩下两科趁状态好明年上半年就能搞定。"

→ 种下种子，不提"续报"二字，但让学员自己想到"我还有两科要报"。

案例 B：铁粉学员转介绍
Input
法规考了75分，特别满意，经常在群里夸
Output

生成自然不刻意的转介绍话术，把"帮介绍"包装成"分享好消息"：

铁粉版："姐，你法规75分真的太牛了！身边有没有同事也想考注安的？你这么有经验，给他们推荐一下咱们班，他们肯定少走弯路。有什么需要的我随时配合你～"

满意版："你这次考得挺不错的，如果有朋友也想考注安，可以一起组队，到时候群里人多点，讨论氛围更好，你学起来也更带劲。"

一般版："最近不少人在问注安班的事，你在咱们班也有一段时间了，要是身边有人想了解，随时让他们加我聊聊，我给安排试听。"

模块七·班主任日常效率工具

朋友圈文案量产→周报自动撰写→学员问题汇总→档案数据清洗。把你每周重复做的事情交给AI。含4个场景+1个动手任务。

工作场景

周五下午4:55，领导在群里说"本周周报别忘了今天交"。你打开空白文档...这周做了什么来着？联系了30个学员、做了5个首访、处理了2个退费意向、发了3条群消息、跟了一次直播答疑。事做了不少，但要写成报告就大脑一片空白。同时下周的朋友圈内容还没准备，"注安备考技巧"这个话题已经写了8遍了，实在想不出新角度。

角色：班主任 · 用AI把自己从重复劳动中解放出来

核心痛点

  • 01周报写作最痛苦——做了很多事但写不出来，写出来像流水账领导不满意
  • 02朋友圈文案内容枯竭——每周都要发学习干货、励志语录，脑子里存货早就用完了
  • 03学员问题无法系统反馈——学员提的零散问题没法整理成给产品和教研的结构化反馈
  • 04学员档案Excel混乱——缺字段、格式不一致、信息过时，手动整理太耗时

提示词模板

模板 7.1：朋友圈/群发文案量产
我是注安培训班的班主任。帮我生成本周社群内容：
本周主题：[如"实务案例分析技巧"]
请输出：
①5条朋友圈文案（不同角度，每条2-3行，有互动引导）
②2条群公告（一条提醒型、一条干货型）
③1条早安语录（和注安备考相关）
避免重复、避免套话。
角色 社群内容创作助手 任务 批量产出朋友圈+群公告+语录 格式 5条朋友圈+2条公告+1条语录 约束 避免重复和套话 输出 可直接复制发布的内容
模板 7.2：周报自动生成
帮我写本周班主任周报：
本周数据：新加学员[X]人，首访完成[X]人，督学联系老学员[X]人，退费意向[X]人，朋友圈[X]条，群消息[X]条
重点事项：[简述2-3件]
遇到问题：[简述]
格式：①关键数据一览 ②本周重点事项 ③问题与风险 ④下周计划
语气：像跟领导口头汇报，不要太正式
角色 周报撰写助手 任务 将零散数据结构化为周报 格式 四段式：数据/重点/问题/计划 约束 语气口语化，不要太正式 输出 可直接提交的周报
模板 7.3：学员问题→产品反馈
以下是近期学员反映的问题：
[粘贴散乱的问题记录]

帮我整理成结构化反馈文档：
①按"课程内容/平台功能/服务质量"分类
②每类标出高频问题（出现2次以上的）
③每个问题附一条改进建议
④输出成表格
角色 产品反馈整理助手 任务 零散问题→结构化反馈文档 格式 三分类+高频标注+改进建议 约束 高频=出现2次以上 输出 表格形式，可直接提交产品/教研
模板 7.4：档案数据清洗
这是我的学员档案数据：
[粘贴Excel数据或上传文件]

帮我检查：
①哪些字段有缺失值（统计每个字段缺失数量和比例）
②哪些数据有异常（如年龄<18或>70）
③哪些格式不一致（如日期格式混乱）

案例演示

案例 A：本周做了什么→周报
Input
新加学员3人，首访完成5人，督学联系老学员30人，退费意向2人，朋友圈3条，群消息12条。重点：处理了2个退费意向都挽留成功；跟了一次张老师直播答疑。问题：有学员反馈技术课第8章讲太快。
Output

关键数据：新加3人｜首访5人｜督学30人｜退费挽留2/2｜朋友圈3条｜群消息12条

重点事项：①2个退费意向全部挽留成功（学员A因工作忙想退，调整了学习节奏；学员B对课程有误解，补充了答疑服务）；②跟了张老师直播答疑，学员参与度不错。

问题与风险：技术课第8章反馈讲太快，已记录，下周和教研反馈。

下周计划：继续完成剩余首访，跟进第8章反馈进展，准备冲刺阶段方案。

案例 B：10条学员吐槽→产品反馈文档
Input
10条散乱的学员反馈（"视频卡顿""第8章太快""APP闪退""题库缺答案""老师讲得好"…）
Output

按课程内容/平台功能/服务质量分类，标注高频问题和改进建议：

课程内容（3条）：第8章太快(×2→高频)→建议拆分为两讲；题库缺答案(×1)→补充解析。

平台功能（4条）：视频卡顿(×2→高频)→技术排查带宽问题；APP闪退(×2→高频)→优先修复；进度不同步(×1)→排查数据同步逻辑。

服务质量（3条）：答疑回复慢(×1)→增加值班答疑时段；老师讲得好(×2)→正向反馈，可做口碑素材。
